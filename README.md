# Void-Tech (ecommerce)

## Designed by Lab2486 a.k.a Lucas Aballay

Este es el proyecto final del curso dictado por CoderHouse de React Js de la carrera Front-End. Se trata sobre un Ecommerce, el cual est√° creado con Vite. Se implementa React.js con React Router, Typed JS, Firebase.
